В файлі render.js знаходяться такі функції:
1. createDOMElement - функція приймає в якості параментів тег, класи, об'єкт з атрибутами де ключ це назва атрибута в його значення це значення атрибута, текс. Функція повертає новострорений об'єкт з необхідними параметрами

2. createCardsGrid - функція приймає в параметри масив об'єктів карточок та селектор куди вставити карточки. Функція створює лист карточок товару в залежності від того скільки було об'єктів передано в масив

3. getData - функція яка в собі містить fetch запрос який отримує файли не з сервера а з папок data

При натисканні на карточку відбувається передача id товару по url. В файлі renderProduct.js ми отримуємо це id за допомогою new URLSearchParams(window.location.search). Потім за допомогою fetch ми отримуємо список товарів та за допомогою функції find() знаходимо id потрібного нам товару. Потім підставляємо отримані дані в розмітку

Етапи передачі товару по сайту
1. Рендер товару на сторінці каталогу.
2. В карточку товару ми передаємо по URL id товару.
3. На сторінці товару ми через new URLSearchParams() знаходимо передану інформацію по URL
4. Робимо запрос через fetch для отримки списка товарів. Знаходимо по id необхідний нам товар.
5. Рендеремо сторінку з тією інформацією яку ми отримати від fetch
6. При натисканні на кнопку купити чи додати в кошик додаємо id товару та кількість товару в localStorage у вигялді об'єтка попередньо використавши JSON.stringify()

Передача товару з сторінки корзину на сторінку оформення відбувається за допомогою передачі по URL. Передаємо сформований об'єкт де є інформацію про назвуб кількість та ціну товару. 

УВАГА!!! Через URL не можна передавати особисту інформацію. І корзину також ліпше не передавати через URL так як він має обмеження і користувач може змінити дані які ми передали. Ліпше використовувати id з localStorage та робити запрос на сервер і отримувати список товару а потім фільтрувати
